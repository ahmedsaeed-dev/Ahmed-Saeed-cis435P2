CIS 435 - Web Technology

3 Credit Hours, Fall 2019
Lectures: Thursday, 6:00 - 8:45 PM, HPEC 1181

Contact Information:

Prof. Jinhua Guo
Office Location: 211 CIS
Phone Number: (313) 583-6439
Email: jinhua@umich.edu

Course Description:

The study of technologies used to design and implement web-based applications. Topics include HTML, CSS, JavaScript, PHP, Rich Internet Application technologies (AJAX), web servers, back-end database connectivity (MySQL), web security, multimedia, XML, JSON, and web services.

Prerequisites:

Previous or concurrent enrollment in CIS 375 or CIS553 or equivalent.

Assignment Summaries:

Project 01: Create a web page that covers topics discussed in chapters 2 through 8. Create a link to your current courses including this one, CIS 435. List some of your hobbies, and include a picture of yourself.

Project 02: Write a web page that enables the user to play the game of 15. There is a 4-by-4 board (implemented as an HTML table) for a total of 16 slots. One of the slots is empty. The other slots are occupied by 15 tiles, randomly numbered from 1 through 15. Any tile next to the currently empty slot can be moved into the currently empty slot by clicking on the tile.

Project 03: Create a PHP and MySQL project that takes 36 total students and devides them up into 6 different one-hour long time slots. tile.

Term Project: Students taking this course will complete a term project in small groups. The project will allow you to put some of the concept introduced in the lectures into practical use. The project is a big part of the course and accounts for 35% of the final grade. You will be required to turn in a project proposal, give two presentations on your project, and turn in a progress report and a final report.
